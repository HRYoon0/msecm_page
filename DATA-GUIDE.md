## 테스트 방법

### 디지털 포트폴리오 테스트

1. https://msecm-page.vercel.app/portfolio.html 접속
2. 닉네임 입력 (예: "테스트학생") → "시작하기" 클릭
3. **프로그램 탭**: 원하는 프로그램 "참여하기" 클릭
4. **내 기록 탭**: 참여한 프로그램 선택 → 날짜/제목/내용 입력 → "기록 저장"
5. 산출물 업로드: 파일 선택 → "업로드"
6. **뱃지 탭**: 기록 1개 작성 시 🌟 첫 발걸음 자동 획득 확인
7. 뱃지 획득 시 축하 모달(콘페티) 표시 확인
8. 우측 상단 🔔 종 아이콘 → 알림 확인

### 교육자료실 테스트

1. https://msecm-page.vercel.app/resources.html 접속
2. 검색바에 "AI" 또는 "수학" 입력 → 실시간 필터링 확인
3. 좌측 카테고리 필터 (수업지도안, 활동지 등) 클릭
4. 과목 필터 (초등, 중등-수학 등) 선택
5. **강사 로그인**: 우측 상단 "강사 로그인" → MSECM / 1234
6. 로그인 후 자료 업로드 폼 → 제목/설명/카테고리 입력 → 업로드
7. 업로드한 자료가 목록에 추가되는지 확인

### 원격 연수원 테스트

1. https://msecm-page.vercel.app/remote-academy.html 접속
2. 강좌 카드 둘러보기 → 카테고리 필터 (AI·디지털, 수학 등)
3. 강좌 카드 클릭 → 상세 모달 (영상 영역, 차시 목록)
4. 우측 상단 "로그인" → 닉네임 입력
5. 로그인 후 "수강 시작" 클릭 → 차시 체크박스 클릭하면 진도율 증가
6. "나의 학습" 메뉴 → 수강 중 강좌, 진도율 확인
7. 전체 차시 완료 시 "수료증" 탭에서 수료 인증 확인
